Первый публичный релиз.

Ключевые особенности версии:
• Встроенный браузер с перехватом URL загрузок и переходом во внешние приложения
• Указание имени файла и подкаталога для сохранения
• Настраиваемые одновременные загрузки
• Скачивание файлов в фоне
• Пакетные загрузки посредством параметров из выбранного JSON-файла
• Оповещение о процессе в области уведомлений по желанию
• Возможность указания различных параметров для запроса
• Поддержка загрузок и выгрузок файлов больших размеров
• Повтор неудавшихся загрузок при появлении сети
• Опция загрузки только при наличии Wi-Fi
• Автоматическое возобновление прерванных загрузок после перезахода в приложение
